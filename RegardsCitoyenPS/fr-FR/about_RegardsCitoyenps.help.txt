TOPIC
    about_RegardsCitoyenPS

SHORT DESCRIPTION
    Ce module powershell permet d'analyzer les données de la vie politique Française, mis a disposition par "Regards Citoyen".
LONG DESCRIPTION
    Les données de la vie politique Française sont mis à disposition à tous les citoyens Français via les site webs suivants:
    www.assemblee-nationale.Fr
    www.senat.Fr

    Toutes les données sont mis à disposition sous leur forme de présentation finale, et sont uniquement bon pour un affiche. 
    Les données mis à disposition ne sont donc pas facilement cumulable, analysable de manière automatisée.
    RegardsCitoyen à créer le site www.nosdeputes.fr qui fait un synthèse globale de ce que nous mis à disposition l'état, et y éffectue une analyse plus étendue.
    Le site www.NosDeputes.fr met les données qu'ils ont extrait des différent site webs également à disposition via un API (Application Programming Interface).

    Le module powershell RegardsCitoyenPS ce base sur l'API mis à disposition par RegardsCitoyen, sauf pour la liste des cmdlets suivantes:
    -> N/A

        Etant donnée qu'un dossier implique beaucoup de sous objets, et afin de réduire l'impact sur la bande passante,
        pas toutes les données sont immédiatement disponible via la recherche simple.

        Afin de récuprer tous les information d'un dossier en particulier il faut utiliser les paramètre -ID et -Full :

        Get-RCDossier -id 2025 -Full
